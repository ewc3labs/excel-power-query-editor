## üöÄ Quick Start

### 1. Install

- Open VS Code ‚Üí Extensions (`Ctrl+Shift+X`)
- Search for **"Excel Power Query Editor"**
- Click **Install**

### 2. Extract & Edit

- Right-click any Excel file ‚Üí **"Extract Power Query from Excel"**
- Edit the generated `.m` file using full VS Code features

### 3. Enable Sync

- Right-click the `.m` file ‚Üí **"Toggle Watch"**
- Your changes are automatically synced to Excel on save
- Built-in backup protection keeps your data safe
